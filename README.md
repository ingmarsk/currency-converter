# Currency Converter :currency_exchange:


## Dependencies

|  Dependency | Version |
| ------------ | ------------ |
|  Ruby | 3.0.0  |
|  Rails | 6.1.6  |
| Bundler |2.2.3|


## Setup

`$ bundle install`

`$ db:setup`

`$ rails s` (http://localhost:3000/)
